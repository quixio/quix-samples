# Real-time dashboard

[This project](https://github.com/quixio/quix-samples/tree/main/python/destinations/Realtime-Dashboard) is a template showing how to build a real-time dashboard with Dash.

## How to run

Create a [Quix](https://portal.platform.quix.ai/self-sign-up?xlink=github) account or log-in and visit the Samples to use this project.

Clicking `Edit code` on the Sample, forks the project to your own Git repo so you can customize it before deploying.

## Environment variables

The code sample uses the following environment variables:

- **input**: Name of the input topic to listen to.
- **output**: Name of the output topic to write to.
- **storage_version**: Version of state. Bump up this variable to reset the state..
